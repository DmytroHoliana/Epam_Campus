1. Нужно убрать пробелы, запятые, и т.д.
Оставляем только буквы и цифры методом → isalnum()
То есть "A man, a plan, a canal — Panama!" нужно сравнивать как "amanaplanacanalpanama".

Переводим в один регистр → .lower(), чтобы все были с маленькой буквы и не было ошибки по не совпадению из-за капслока

2. Цикл просмотра букв 1 и -1, до средины, реверс нельзя, поэтому нужно создать такой, который просматривает до средины и останавливается, если неправильно и выдает False



def is_palindrome(s: str) -> bool:
    # 1. Приводим строку к нижнему регистру
    s = s.lower()
    
    # 2. Оставляем только буквы и цифры
    clean = "" # создание пустой строки clean 
    for c in s:	# с - переменная в которую переносится текст
        if c.isalnum(): 	убирается лишнее
            clean += c		идёт добавление посимвольно

    # 3. Проверяем строку с двух сторон
    left = 				начинаем с 0, т.е. 1й символ
    right = len(clean) - 1		начинаем с -1, т.е. последний 
    
    while left < right:			
        if clean[left] != clean[right]:  если лево не равно право
            return False
        left += 1			если равно, продолжается подсчёт
        right -= 1
    
    return True
